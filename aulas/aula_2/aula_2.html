Inf representa valores infinitos, pode ser +Inf ou -Inf

NaN Not a Number, resultado de operaes matemticas impossveis

factor especifica que um conjunto de valores representa os nveis de um fator, usado para anlises de dados categricos

Operaes lgicas e comparaes

O R permite fazer todo tipo de comparao e teste lgico, abrindo possibilidade de selecionar itens para indexao baseados em diversas condies. os objetos testados podem ser de qualquer tipo

Testes lgicos retornam valores TRUE ou FALSE
== a comparao de igualdade. Leia como igual a?
!= a comparao de desigualdade. Leia como diferente de?

Comparaes de igualdade com ponto flutuante

Nmeros decimais so representados internamente como floating point, e a preciso varia de mquina para mquina. Veja mais detalhes em http://floating-point-gui.de/

i = 0.1
i = i + 0.05
i == 0.15
## [1] FALSE

Resumidamente, se voc quiser comparar com preciso a igualdade de operaes com decimais, use a funoo all.equal(). Para comparaes do tipo maior/menor, no preciso se preocupar (muito)

i = 0.1
i = i + 0.05
all.equal(i, 0.15)
## [1] TRUE

Outras comparaes

a <= b a menor ou igual a b?
a >= b a maior que ou igual a b?
!x no x

Operadores lgicos e if()

O R tambm pode testar duas condies ao mesmo tempo, com os operadores E e OU
a & b retorna TRUE se tanto a quanto b forem verdade
a | b retorna TRUE se pelo menos um, a ou b, for verdade

teste = c(1, 2, 2, 3, 4, 5, 6, 7)
(teste[1] == 1) & (teste[2]/2 == 1)
## [1] TRUE
(teste[1] == 1) & (teste[2]/3 == 1)
## [1] FALSE
(teste[2] == 1) | (teste[2]/2 == 1)
## [1] TRUE

Fazendo a mesma coisa vrias vezes

se precisamos repetir uma tarefa vrias vezes, usamos o lao (loop)

for (i in 1:10) {
    a = i^2
    message(a)
}

i chamado de ndice. Pode ser qualquer letra ou expresso, utilizado para contar quantas vezes o lao foi executado
1:10 1 o valor inicial de i, 10 o valor final.

(i in 1:10) significa comece com i = 1, e v aumentando de um em um at 1 = 10
O bloco entre {} ser executado enquanto a condio for verdadeira
